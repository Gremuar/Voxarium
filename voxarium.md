# Voxarium — технический план разработки приложения

## 1. Общая информация о проекте

Название проекта: **Voxarium**

Тип проекта:
Локальное приложение для преобразования текста в естественную речь (Text-To-Speech, TTS) с возможностью создания аудиокниг, озвучки документов и управления голосовыми моделями.

Основная идея:
Создать полноценное локальное приложение, которое позволяет пользователю:

- загружать текстовые документы;
- очищать и подготавливать текст;
- разбивать большие документы на логические части;
- синтезировать речь локальными нейросетевыми моделями;
- выбирать голоса;
- сохранять результат в аудиофайлы;
- создавать полноценные аудиокниги;
- управлять процессом генерации;
- работать полностью автономно без отправки данных в облако.

Главный приоритет проекта:

1. Качество звучания.
2. Естественность голоса.
3. Стабильность работы.
4. Поддержка русского языка.
5. Возможность расширения функционала.

## 3.1. Движок синтеза речи

Основной движок:

Причины выбора:

- высокое качество русского языка;
- поддержка клонирования голоса;
- локальная работа;
- хорошее качество интонации;
- возможность дальнейшего обучения.

Архитектура должна позволять заменить движок:

Например:

- Bark;
- StyleTTS2;
- OpenVoice;
- другие модели.

Для этого необходимо создать абстрактный интерфейс:
SpeechEngine

|
|-- FutureEngine

    3

Функции:

- загрузка проекта;
- сохранение состояния;
- продолжение после перезапуска;

# 5.3. Очистка текста

Перед передачей в TTS текст должен проходить обработку.

Функции:

- удаление лишних пробелов;
- исправление переносов;
- нормализация кавычек;
- обработка тире;
- обработка сокращений;
- замена сложных конструкций.

Пример:

Было:


Гл. 1.
—Здравствуйте...


После обработки:


Глава первая.

Здравствуйте.


---

# 5.4. Разделитель текста

Большие книги нельзя отправлять одним запросом.

Необходимо:

Разбивать:

- по главам;
- по абзацам;
- по смысловым блокам.

Ограничения:

Например:


максимум:
500-1000 символов


Алгоритм:

1. Найти границы предложений.
2. Не разрывать слова.
3. Не разрывать диалоги.
4. Сохранять порядок.
