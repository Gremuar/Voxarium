# Voxarium

**Voxarium** — модульная платформа синтеза речи с поддержкой различных TTS-бэкендов.

## Основные возможности

- Поддержка нескольких TTS-бэкендов
- Управление голосами
- Клонирование голоса
- Очередь озвучивания
- Работа с проектами
- Поддержка PDF, DOCX, Markdown, TXT
- Полностью локальная работа
- Современный веб-интерфейс

## Планируемые бэкенды

- Edge TTS
- XTTS v2
- Piper
- Fish Speech
- CosyVoice
- Kokoro
